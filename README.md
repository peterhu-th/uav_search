# UAV Search - 基于贝叶斯热力图的无人机协同搜索路径规划

本项目模拟了多无人机在给定海域内搜索移动目标的任务，结合贝叶斯滤波进行目标概率预测，并利用人工势场实现无人机路径规划与协同。

## 1. 目录结构与文件作用

### `src/`
* **`environment.py`**: 环境模型核心。负责维护 2D 概率网格，处理真实目标的机动逻辑，并调用 C++ 底层进行概率更新。
* **`uav_controller.py`**: 无人机集群控制器。实现人工势场，计算引力和斥力，生成无人机的移动指令。
* **`bayes_core.cpp`**: C++ 实现的底层计算引擎。包含高斯扩散和雷达扫描的高性能计算逻辑。
* **`plot_utils.py`**: 绘图工具，用于生成仿真轨迹图和热力图可视化。

### `lib/` (二进制库)
* **`bayes_core.dll` / `bayes_core.so`**: 由 `bayes_core.cpp` 编译而成的动态链接库，供 Python 调用以提升计算性能。

### `config/`
* **`config.py`**: 全局参数配置文件。包含海域范围、无人机性能、雷达参数、APF 系数及 Monte Carlo 仿真设置。

## 2. 核心算法说明

1.  **贝叶斯滤波 (Bayesian Filtering)**:
    * **预测步**：使用高斯核对概率地图进行扩散，模拟目标随时间推移的位置不确定性。
    * **观测步**：当无人机雷达扫描某区域未发现目标时，降低该区域的后验概率（降温），并重新归一化全图概率。
2.  **人工势场法 (APF)**:
    * **引力**：无人机受概率地图中高概率点的吸引。
    * **斥力**：无人机之间存在斥力以防止碰撞并实现搜索区域的分离（阵型展开）。
3.  **目标机动策略**: 支持随机游走、直线航行以及躲避无人机的逃逸策略。

## 3. 仿真任务介绍

项目包含三个预设的仿真脚本，可通过命令行运行：

### 基础仿真演示 (`demo.py`)
演示单次搜索过程，实时可视化显示概率热力图、无人机轨迹及真实目标位置。
* **运行命令**: `python tasks/demo.py`

### 搜索效能评估 (`search_time.py`)
通过蒙特卡洛仿真计算特定数量（默认为 2 架）无人机在当前配置下的平均搜索用时和成功率。
* **运行命令**: `python tasks/search_time.py --save true --collapse true`
* **参数**:
    * `--save`: 是否保存仿真轨迹图。
    * `--collapse`: 是否启用信任度崩塌策略。

### 规模优化搜索 (`min_uavs.py`)
在给定的任务时限内，通过递减无人机数量，寻找满足成功率要求（如 50%）的最少无人机配置。
* **运行命令**: `python tasks/min_uavs.py --save true`

## 4. 环境配置要求

### 软件依赖
* **Python 3.10+**
* **C++ 编译器**: 若需重新编译 `lib/` 中的库，需支持 C++11 的编译器（如 GCC 或 MSVC）。

### Python 库需求
* `numpy`: 矩阵运算与概率地图处理。
* `matplotlib`: 仿真过程可视化及结果绘图。
* `ctypes`: Python 与 C++ 动态库的交互（标准库）。

### 硬件建议
* 由于涉及大规模网格计算和蒙特卡洛多线程运行建议，推荐使用 8GB 以上内存。
